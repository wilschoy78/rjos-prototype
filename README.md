# RJO Industrial Products Prototype

A Vite + React single-page prototype for RJO Industrial Products.

## Scope

- Role-based operational workspace
- Master data management
- Sales operations and quotations
- Procurement and goods receiving
- Inventory operations and stock transfers

## Tech Stack

- React
- Vite
- Tailwind CSS
- lucide-react

## Local Development

```bash
cd /Users/wilson/Desktop/2026/RJO/rjos-project/rjos-prototype
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
